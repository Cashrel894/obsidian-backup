#CS188 
**机器学习**(Machine Learning)，是一门指根据给定的数据，从中建构或学习特定模型参数的学科。

机器学习包含的算法各式各样，分别处理不同类型的问题以及不同类型的数据。

机器学习算法可以分为两大类：
- **监督学习**(Supervised Learning)：通过推断输入数据和输出数据之间的关系，从而预测全新输入的输出结果。
- **无监督学习**(Unsupervised Learning)：在无对应输出数据的情况下，通过只学习输入数据，来识别输入数据的内在结构、组织方式等。

机器学习算法学习的数据集通常分为几个部分：
- 训练数据 (Training Data)
- 验证数据 (Validation Data, a.k.a hold-out or Development Data)
- 测试数据 (Testing Data)

在学习数据时，通常需要通过调整 **超参数**(Hyperparameters) 或更换机器学习算法来试图达到最优效果。
