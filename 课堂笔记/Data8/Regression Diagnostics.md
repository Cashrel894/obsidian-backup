为了确认一个回归是否足够好，我们可以进行***回归诊断(Regression Diagnostics)***。通常而言，回归诊断需要可视化某些统计量，进而用直观的方式判断回归的好坏。

## Visual Diagnostics
例如我们可以直接可视化回归直线：
![[../../附件/Pasted image 20251225115008.png]]

但更常见的是可视化***残差(Residual)***：
![[../../附件/Pasted image 20251225115036.png]]

一个好的回归，它的残差图像不会显示出明显的统计学趋势，且**残差的分布基本上关于回归直线对称**。否则，或许其他方式的拟合（如非线性）可以更好地刻画变量间的关系。

同时，使用残差图像有时还可以发现数据的***异方差性(Heteroscedasticity)***。即，y的分散程度并不均匀，而是关于 x 有着明显变化。例：![[../../附件/Pasted image 20251225115851.png]]

此时，回归的准确性也并不均匀，而是关于 x 有一定变化。
