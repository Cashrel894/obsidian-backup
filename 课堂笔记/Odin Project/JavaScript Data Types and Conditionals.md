## Strings 
### Template Literals
被反引号 \` 包裹的字符串称作模板字面量。

在模板字面量中：
- 可以用 `${[expression]}` 形式内嵌 JS 代码，\[expression\] 中的代码会被计算并转换为字符串，插入到字符串的对应位置。如 \`1 + 2 = ${1 + 2}\`等价于字符串"1 + 2 = 3"。
- 同时，模板字面量也允许多行字符串。
